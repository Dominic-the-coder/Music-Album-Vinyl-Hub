# Music Store App

A simple Android application built with Kotlin that allows users to browse, search, and purchase music albums.

## Features

* View album list
* Search albums by title
* Filter albums by category
* Add albums to cart
* View cart summary
* Local data storage using Room Database

## Installation

1. Clone the repository

```bash
git clone (https://github.com/Dominic-the-coder/Music-Album-Vinyl-Hub#js-repo-pjax-container)
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the application on an emulator or physical device

## Requirements

* Android Studio Narwhal or later
* Android SDK 24+
* Kotlin 2.x

## How to Use

1. Launch the application
2. Browse available albums
3. Use the search bar to find albums
4. Add albums to your cart
5. Proceed to checkout
